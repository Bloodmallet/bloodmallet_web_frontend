Talents

This chart shows how much power each talent provides via some stats for each. Please read the Disclaimer section carefully.

It uses the "tier appropriate" SimulationCraft profile. If the fight style is neither patchwerk nor castingpatchwerk a "tier appropriate" fallback profile is used to improve the data quality. In a custom chart if the creator provided a profile, it replaces the formerly mentioned profiles.

Stats are calculated by comparing a profile with a talent combination containing the specific talent to the same profile that uses no talent in that row.

Stats
  • Mean: This is the mean contribution of this talent for all existing talent combinations with this talent.
  • Min: This is the minimal contribution of this talent found in all existing talent combinations with this talent. When hovering with your cursor over this line you'll see the talent combination. In this talent combination this talent is at its weakest performance.
  • Max: This is the highest contribution of this talent found in all existing talent combinations with this talent. When hovering with your cursor over this line you'll see the talent combination. In this talent combination this talent is at its peak performance.
  • Max dps: This is the contribution to the highest dps talent combinations found in all existing talent combinations with this talent. When hovering with your cursor over this line you'll see the talent combination. This talent combination is the strongest of all talent combinations this talent is part of.
  • Max dps minus global best: This is the minimal distance to the strongest talent combination. E.g. if the best setup for the talent of your choice is 2% weaker than the optimal talent combination, you'll see here a "-2.00%". Zero percent is the best possible value here. When hovering with your cursor over this line you'll see the talent combination. This talent combination is the strongest of all talent combinations this talent is part of. You see how much weaker it is compared to the strongest one.
